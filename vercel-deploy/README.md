# Flying Car Game

A 3D flying car game built with Three.js featuring multiple cars, day/night cycles, and physics-based movement.

## Features

- Three different cars to choose from
- Day/night cycle with dynamic lighting
- Physics-based movement and controls
- Speed boost system
- Altitude control
- Score tracking
- Realistic city environment with buildings and trees
- Dynamic camera controls

## Controls

- ↑/↓: Move Forward/Back
- ←/→: Turn Left/Right
- W/S: Thrust Up/Down
- Q/E: Tilt Left/Right
- SHIFT: Speed Boost
- Mouse Wheel: Zoom Camera

## Development

1. Clone the repository
2. Install dependencies: `npm install`
3. Start local server: `npm start`
4. Open `http://localhost:3000` in your browser

## Deployment

This game is configured for deployment on Vercel. To deploy:

1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the project directory
3. Follow the prompts to deploy

## Technologies Used

- Three.js for 3D graphics
- HTML5/CSS3 for UI
- JavaScript for game logic
