# 🚗 Flying Car Game

A dynamic 3D flying car game built with Three.js, featuring an interactive city environment with animated characters, flying birds, and dynamic weather effects.

![Flying Car Game Demo](demo.gif)

[Watch Demo Video](https://youtu.be/OyZCZQoFZhE?si=Yig1PHql7cVETgVU)

## 🎮 Features

### Vehicle & Controls
- 3 unique flying cars with different colors and styles:
  - 🏎️ Red Racer
  - 🚙 Blue Cruiser
  - 🚓 Green Patrol
- Smooth flight controls with physics-based movement
- Boost system with particle effects
- Real-time speedometer and altitude tracking
- Dynamic camera with zoom functionality

### Environment
- 🌆 Procedurally generated city with buildings
- 🌳 Dynamic environment with trees and vegetation
- ☁️ Animated clouds system
- 🦅 Flying birds with realistic movement
- 🌞/🌙 Day/Night cycle with lighting changes
- 🌫️ Atmospheric fog effects

### Characters & Animation
- 🚶‍♂️ Walking pedestrians with realistic animations
- 🐕 Animated animals roaming the streets
- Smart pathfinding to avoid obstacles
- Natural movement patterns and behaviors

### UI/UX
- Modern, game-style interface
- Real-time stats display (Speed, Altitude, Score)
- Boost meter with visual feedback
- Intuitive control scheme
- Responsive design for all screen sizes

## 🎯 Controls

- **Movement**:
  - ↑/↓ - Move Forward/Backward
  - ←/→ - Turn Left/Right
  - W/S - Thrust Up/Down
  - Q/E - Tilt Left/Right
  - SHIFT - Speed Boost
  - 🖱️ Mouse Wheel - Zoom Camera

## 🛠️ Technical Stack

- Three.js for 3D rendering
- Custom physics engine for flight mechanics
- Ruby on Rails backend (v3.2.2)
- Modern JavaScript (ES6+)
- HTML5 & CSS3 with advanced animations

## 🚀 Getting Started

1. **Prerequisites**
   ```bash
   ruby -v  # Requires Ruby 3.2.2
   ```

2. **Installation**
   ```bash
   git clone https://github.com/yourusername/flying-car-game.git
   cd flying-car-game
   bundle install
   ```

3. **Running the Game**
   ```bash
   rails server
   ```
   Visit `http://localhost:3000` in your browser
